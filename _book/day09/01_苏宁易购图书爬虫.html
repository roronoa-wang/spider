课程整体概要

苏宁易购图书爬虫

需求

  • 需求: 提取苏宁易购中电子书信息
  • 起始URL(已废): http://snbook.suning.com/web/trd-fl/999999/0.htm
    • 新起始URL: https://book.suning.com/
  • 提取: 大分类:名称,URL; 小分类名称,URL; 图书的标题,图片,出版商,价格信息

思路:

  • 生成suning项目, 生成图书爬虫
  • 准备起始URL
  • 在parse函数,解析数据,提取大分类标题和URL,小分类标题和URL, 通过小分类URL构造请求,获取小分类的图书列表
  • 写一个解析函数,解析小分类中电子书列表,提取: 图书的标题,图片
  • 构造详情页请求, 获取出版社信息
  • 找出详情页URL与价格URL直接规律了, 构建价格请求, 获取价格数据
  • 实现翻页效果

实现:

  1. 生成suning项目: scrapy startproject suning
  2. 生成suning图书爬虫: 
     cd suning
 scrapy genspider book suning.com

     class BookSpider(scrapy.Spider):
 name = 'book'

 # allowed_domains 不要写成: book.suning.com
 # 否则: https://list.suning.com/1-264003-0.html, https://product.suning.com/0070121210/103281921.html
 # 都会被过滤掉
 allowed_domains = ['suning.com']

 # 起始URL
 start_urls = ['https://book.suning.com/']

 def parse(self, response):
     """提取大分类和小分类"""
     # 获取包含大分类信息的dl列表
     dls = response.xpath('//div[@class="menu-item"]/dl')

     # 弹出层的divs
     sub_divs = response.xpath('//div[contains(@class,"submenu-left")]')

     # 遍历dls,提取大分类
     for index, dl in enumerate(dls):
         # 用于记录提取到数据
         item = {}
         item['b_category_name'] = dl.xpath('./dt/h3/a/text()').extract_first()
         item['b_category_url'] = dl.xpath('./dt/h3/a/@href').extract_first()

         # 获取小分类a标签列表
         a_s = dl.xpath('./dd/a')
         # 如果没有小分类, 我们提取弹出层上小分类
         if len(a_s) == 0:
             sub_div = sub_divs[index]
             a_s = sub_div.xpath('.//a')
         # 提取小分类信息
         for a in a_s:
             item['s_category_name'] = a.xpath('./text()').extract_first()
             item['s_category_url'] = a.xpath('./@href').extract_first()
             # print(item)
             # 进入到小分类列表页
             # 根据小分类的URL, 构建请求
             yield scrapy.Request(item['s_category_url'], callback=self.parse_list, meta={'item': deepcopy(item)})

      def parse_price(self, response):
  """解析价格信息"""
  item = response.meta['item']
  # 有的商品有推广价格, 那么此时页面展示就是推广价格; 说明: 一般套装书使用的是推广价格
  # 我们先尝试提取推广价格
  # response.text: 响应文本字符串
  promotionPrice = re.findall('"promotionPrice":"(.+?)"', response.text)
  # 如果有推广价格, 就使用推广价格, 作为图书价格
  if promotionPrice and len(promotionPrice) != 0 and promotionPrice[0].strip() != '':
      item['price'] = promotionPrice[0]
  else:
      # 如果没有推广价格, 就使用netPrice做为价格  
      item['price'] = re.findall('"netPrice":"(.+?)"', response.text)[0]
  yield item

    • 注意:

      • 根据渲染后面的Element我们不能获取价格数据, 因为价格数据是通过js生成的

      • 我们通过详情页URL和价格URL规律, 来通过详情URL来生成价格URL

  8. 实现小分类图书列表翻页

    • 在 parse_book_list 中增加构建下一页请求的URL

      def parse_list(self, response):
  ...

  # 列表下一页获取
  # 获取当前页的页号
  current_page = int(re.findall('param.currentPage = "(\d+)";', response.text)[0])
  # 获取总的页码数
  page_numbers = int(re.findall('param.pageNumbers = "(\d+)";', response.text)[0])
  # 计算下一页的页号
  next_page = current_page + 1
  # 如果下一页的页号小于总页码数, 就说明有下一页, 那么就根据这个页号生成下一页的URL
  if next_page < page_numbers:
      # 通过小分类进入的URL为格式: https://list.suning.com/1-264003-0.html 
      # 我们需要做的就是使用 -下一页号.html 替换掉: -0.html
      # 生成  -下一页页号.html 
      s = '-{}.html'.format(next_page)
      # 把  -下一页页号.html 替换掉当前页中 -当前页.html; 为了能够匹配我们使用了正则表达式来进行替换
      next_url = re.sub('-\d+.html', s, response.url)
      #print(next_url)
      yield scrapy.Request(next_url, callback=self.parse_list, meta={'item': deepcopy(item)})
